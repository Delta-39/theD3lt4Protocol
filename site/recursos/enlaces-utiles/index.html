    Enlaces Útiles

    En esta sección encontrarás enlaces a recursos útiles para mejorar tus habilidades en hacking ético y ciberseguridad.

    Plataformas de Aprendizaje

    • TryHackMe - Plataforma interactiva para aprender ciberseguridad con laboratorios prácticos.
    • Hack The Box - Retos y laboratorios de hacking para desarrollar habilidades.
    • OverTheWire - Wargames para aprender conceptos básicos de seguridad informática.
    • Cybrary - Cursos en línea gratuitos sobre ciberseguridad.

    Blogs y Recursos Técnicos

    • SecLists - Colección de listas útiles para pentesting, como diccionarios y listas de passwords.
    • Exploit Database - Base de datos de exploits y vulnerabilidades.
    • The Art of Hacking - Coleccion de recursos, creado por Omar Santos.
    • Conocimientos Obsidian - Coleccion de varios recursos, incluido programacion,docker,hacking, redes,etc

    Comunidades y Foros

    • Reddit - NetSec - Comunidad de noticias y discusión sobre ciberseguridad.
    • Null Byte - Guías y tutoriales sobre hacking ético y ciberseguridad.
    • Ciberseguridad en Español - Comunidad en español dedicada a la seguridad informática.

    Red Team

    • Rev Shell Generator -
    • GTFOBins

    Blue Team

    • CIS Benchmarks - Hardening de S.O.
    • Bandit - Herramienta para auditar codigo python.
    • OWASP TOP 10 - Top 10 ataques a aplicaciones web.

    Frameworks

    • NIST - Marco de ciberseguridad del NIST para mejorar la gestión de riesgos en organizaciones.
    • MITRE ATT&CK - Conjunto de tácticas y técnicas utilizadas en ataques reales, clasificadas por MITRE.
    • Cyber Kill Chain - Modelo que describe las etapas de un ataque cibernético, desde el reconocimiento hasta la explotación.
    • MITRE D3FEND - Base de datos de contramedidas defensivas para diferentes ataques cibernéticos.

    OSINT

    • Way Back Machine - Herramienta para ver versiones archivadas de sitios web a lo largo del tiempo.
    • Crt - Servicio para buscar certificados TLS/SSL emitidos para dominios, útil en investigaciones de OSINT.
    • Google Exploit DB - Base de datos con dorks de Google para encontrar información sensible en la web.
    • The OSINT Framework - Framework interactivo que organiza una amplia gama de herramientas y recursos OSINT
